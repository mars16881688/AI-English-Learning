# Lesson 01 Vocabulary Cards

> 复习方法：遮住右边，看英文说中文；再遮住左边，看中文说英文

## Core Words （核心单词）

### artificial /ˌɑːrtɪˈfɪʃəl/
- 中文：人工的，人造的
- 词根：artifice（技巧）+ -ial（形容词后缀）
- 例句：Artificial intelligence can process data much faster than humans.
- 搭配：artificial intelligence, artificial neural network

### intelligence /ɪnˈtelɪdʒəns/
- 中文：智能，智力
- 相关词：intelligent（adj. 聪明的），intelligently（adv.）
- 例句：The test measures different types of intelligence.
- 搭配：artificial intelligence, human intelligence, intelligence test

### machine learning
- 中文：机器学习
- 例句：Machine learning allows computers to learn from data.
- 注意：这是一个固定术语，不要拆开理解

### deep learning
- 中文：深度学习
- 例句：Deep learning uses neural networks with many layers.
- 区分：machine learning（机器学习）是大类，deep learning（深度学习）是其子集

### neural network /ˈnʊrəl ˈnetwɜːrk/
- 中文：神经网络
- 词根：neural（神经的）+ network（网络）
- 例句：Neural networks are inspired by the structure of the human brain.
- 搭配：artificial neural network, convolutional neural network

### algorithm /ˈælɡərɪðəm/
- 中文：算法
- 例句：A sorting algorithm arranges data in a specific order.
- 搭配：machine learning algorithm, search algorithm

### data /ˈdeɪtə/
- 中文：数据
- 注意：正式英语中 data 是复数（datum 的复数形式），但日常用法中常当单数用
- 例句：AI models need large amounts of data to learn effectively.
- 搭配：training data, data set, data processing

### model /ˈmɑːdl/
- 中文：模型
- 在 AI 语境中：指训练好的程序，能做预测或生成内容
- 例句：GPT is a large language model trained on text data.
- 搭配：language model, AI model, train a model
